VkValidationFeaturesEXT.prototype.pNext null is null or a reference to an extension-specific structure. VkValidationFeaturesEXT.prototype.enabledValidationFeatureCount Number is the number of features to enable. VkValidationFeaturesEXT.prototype.pEnabledValidationFeatures Int32Array is an array of VkValidationFeatureEnableEXT values specifying the validation features to be enabled. VkValidationFeaturesEXT.prototype.disabledValidationFeatureCount Number is the number of features to disable. VkValidationFeaturesEXT.prototype.pDisabledValidationFeatures Int32Array is an array of VkValidationFeatureDisableEXT values specifying the validation features to be disabled.
