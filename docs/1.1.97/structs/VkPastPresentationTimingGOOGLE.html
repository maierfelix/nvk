Properties VkPastPresentationTimingGOOGLE.prototype.presentID Number is an application-provided value that was given to a previous vkQueuePresentKHR command via VkPresentTimeGOOGLE::presentID (see below). It can be used to uniquely identify a previous present with the vkQueuePresentKHR command. VkPastPresentationTimingGOOGLE.prototype.desiredPresentTime Number is an application-provided value that was given to a previous vkQueuePresentKHR command via VkPresentTimeGOOGLE::desiredPresentTime. If non-zero, it was used by the application to indicate that an image not be presented any sooner than desiredPresentTime. VkPastPresentationTimingGOOGLE.prototype.actualPresentTime Number is the time when the image of the swapchain was actually displayed. VkPastPresentationTimingGOOGLE.prototype.earliestPresentTime Number is the time when the image of the swapchain could have been displayed. This may differ from actualPresentTime if the application requested that the image be presented no sooner than VkPresentTimeGOOGLE::desiredPresentTime. VkPastPresentationTimingGOOGLE.prototype.presentMargin Number is an indication of how early the vkQueuePresentKHR command was processed compared to how soon it needed to be processed, and still be presented at earliestPresentTime. Default Properties VkPastPresentationTimingGOOGLE.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkPastPresentationTimingGOOGLE.memoryLayout Object Object describing the memory layout for this structure.
