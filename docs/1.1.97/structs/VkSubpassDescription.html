Parameters object When instantiated with an object, members can be filled directly. Properties VkSubpassDescription.prototype.flags VkSubpassDescriptionFlagBits is a bitmask of VkSubpassDescriptionFlagBits specifying usage of the subpass. VkSubpassDescription.prototype.pipelineBindPoint VkPipelineBindPoint is a VkPipelineBindPoint value specifying the pipeline type supported for this subpass. VkSubpassDescription.prototype.inputAttachmentCount Number is the number of input attachments. VkSubpassDescription.prototype.pInputAttachments Array[VkAttachmentReference] is an array of VkAttachmentReference structures defining the input attachments for this subpass and their layouts. VkSubpassDescription.prototype.colorAttachmentCount Number is the number of color attachments. VkSubpassDescription.prototype.pColorAttachments Array[VkAttachmentReference] is an array of VkAttachmentReference structures defining the color attachments for this subpass and their layouts. VkSubpassDescription.prototype.pResolveAttachments Array[VkAttachmentReference] is an optional array of colorAttachmentCount VkAttachmentReference structures defining the resolve attachments for this subpass and their layouts. VkSubpassDescription.prototype.pDepthStencilAttachment VkAttachmentReference is a reference to a VkAttachmentReference specifying the depth/stencil attachment for this subpass and its layout. VkSubpassDescription.prototype.preserveAttachmentCount Number render pass attachment indices identifying attachments that are not used by this subpass, but whose contents must be preserved throughout the subpass. VkSubpassDescription.prototype.pPreserveAttachments Uint32Array Default Properties VkSubpassDescription.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
