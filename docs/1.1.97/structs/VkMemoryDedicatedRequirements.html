Properties VkMemoryDedicatedRequirements.prototype.sType Number is the type of this structure. VkMemoryDedicatedRequirements.prototype.pNext null is null or a reference to an extension-specific structure. VkMemoryDedicatedRequirements.prototype.prefersDedicatedAllocation Boolean specifies that the implementation would prefer a dedicated allocation for this resource. The application is still free to suballocate the resource but it may get better performance if a dedicated allocation is used. VkMemoryDedicatedRequirements.prototype.requiresDedicatedAllocation Boolean specifies that a dedicated allocation is required for this resource. Default Properties VkMemoryDedicatedRequirements.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkMemoryDedicatedRequirements.memoryLayout Object Object describing the memory layout for this structure.
