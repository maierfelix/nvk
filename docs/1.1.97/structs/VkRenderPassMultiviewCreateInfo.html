Parameters object When instantiated with an object, members can be filled directly. Properties VkRenderPassMultiviewCreateInfo.prototype.sType Number is the type of this structure. VkRenderPassMultiviewCreateInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkRenderPassMultiviewCreateInfo.prototype.subpassCount Number number of view masks, where each mask is a bitfield of view indices describing which views rendering is broadcast to in each subpass, when multiview is enabled. If subpassCount is zero, each view mask is treated as zero. VkRenderPassMultiviewCreateInfo.prototype.pViewMasks Uint32Array VkRenderPassMultiviewCreateInfo.prototype.dependencyCount Number view offsets, one for each dependency. If dependencyCount is zero, each dependency's view offset is treated as zero. Each view offset controls which views in the source subpass the views in the destination subpass depend on. VkRenderPassMultiviewCreateInfo.prototype.pViewOffsets Int32Array VkRenderPassMultiviewCreateInfo.prototype.correlationMaskCount Number is zero or a number of correlation masks. VkRenderPassMultiviewCreateInfo.prototype.pCorrelationMasks Uint32Array is an array of view masks indicating sets of views that may be more efficient to render concurrently. Default Properties VkRenderPassMultiviewCreateInfo.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
