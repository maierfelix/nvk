Properties VkMemoryType.prototype.propertyFlags VkMemoryPropertyFlagBits is a bitmask of VkMemoryPropertyFlagBits of properties for this memory type. VkMemoryType.prototype.heapIndex Number describes which memory heap this memory type corresponds to, and must be less than memoryHeapCount from the VkPhysicalDeviceMemoryProperties structure. Default Properties VkMemoryType.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance.
