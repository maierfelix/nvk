Parameters object When instantiated with an object, members can be filled directly. Properties VkViewport.prototype.x Number VkViewport.prototype.y Number are the viewport's upper left corner [eq]#(x,y)#. VkViewport.prototype.width Number VkViewport.prototype.height Number are the viewport's width and height, respectively. VkViewport.prototype.minDepth Number VkViewport.prototype.maxDepth Number are the depth range for the viewport. It is valid for minDepth to be greater than or equal to maxDepth. Default Properties VkViewport.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkViewport.memoryLayout Object Object describing the memory layout for this structure.
