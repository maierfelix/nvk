 Categories vkCmdClearAttachments Clear regions within bound framebuffer attachments Syntax 
vkCmdClearAttachments();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. attachmentCount Number pAttachments Array[VkClearAttachment] is a pointer to an array of slink:VkClearAttachment structures defining the attachments to clear and the clear values to use. If any attachment to be cleared in the current subpass is ename:VK_ATTACHMENT_UNUSED, then the clear has no effect on that attachment. rectCount Number pRects Array[VkClearRect] points to an array of slink:VkClearRect structures defining regions within each selected attachment to clear.
