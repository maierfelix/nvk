VkResult vkGetPhysicalDeviceCalibrateableTimeDomainsEXT();
Parameters physicalDevice VkPhysicalDevice is the physical device from which to query the set of calibrateable time domains. pTimeDomainCount Object.$(Number) is a reference to a number related to the number of calibrateable time domains available or queried, as described below. pTimeDomains Int32Array is either null or a reference to an array of VkTimeDomainEXT values, indicating the supported calibrateable time domains.
