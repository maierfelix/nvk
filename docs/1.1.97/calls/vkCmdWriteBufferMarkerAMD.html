 Categories vkCmdWriteBufferMarkerAMD Execute a pipelined write of a marker value into a buffer Syntax 
void vkCmdWriteBufferMarkerAMD();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. pipelineStage Number is one of the VkPipelineStageFlagBits values, specifying the pipeline stage whose completion triggers the marker write. dstBuffer VkBuffer is the buffer where the marker will be written to. dstOffset Number is the byte offset into the buffer where the marker will be written to. marker Number is the 32-bit value of the marker.
