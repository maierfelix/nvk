 Categories vkCmdBindShadingRateImageNV Bind a shading rate image on a command buffer Syntax 
void vkCmdBindShadingRateImageNV();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. imageView VkImageView is an image view handle that specifies the shading rate image. imageView may be set to VK_NULL_HANDLE, which is equivalent to specifying a view of an image filled with zero values. imageLayout Number is the layout that the image subresources accessible from imageView will be in when the shading rate image is accessed.
