VkResult vkCreateGraphicsPipelines();
Parameters device VkDevice is the logical device that creates the graphics pipelines. pipelineCache VkPipelineCache is either dlink:VK_NULL_HANDLE, indicating that pipeline caching is disabled; or the handle of a valid <> object, in which case use of that cache is enabled for the duration of the command. createInfoCount Number pCreateInfos Array[VkGraphicsPipelineCreateInfo] is an array of slink:VkGraphicsPipelineCreateInfo structures. pAllocator VkAllocationCallbacks controls host memory allocation as described in the <> chapter. pPipelines Array[VkPipeline] is an array in which the resulting graphics pipeline objects are returned.
