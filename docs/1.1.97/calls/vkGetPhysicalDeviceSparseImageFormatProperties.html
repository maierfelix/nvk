 Categories vkGetPhysicalDeviceSparseImageFormatProperties Retrieve properties of an image format applied to sparse images Syntax 
void vkGetPhysicalDeviceSparseImageFormatProperties();
Parameters physicalDevice VkPhysicalDevice is the physical device from which to query the sparse image capabilities. format VkFormat is the image format. type VkImageType is the dimensionality of image. samples int32_t is the number of samples per texel as defined in VkSampleCountFlagBits. usage VkImageUsageFlagBits is a bitmask describing the intended usage of the image. tiling VkImageTiling is the tiling arrangement of the texel blocks in memory. pPropertyCount Object.$(Number) is a reference to a number related to the number of sparse format properties available or queried, as described below. pProperties Array[VkSparseImageFormatProperties] is either null or a reference to an array of VkSparseImageFormatProperties structures.
