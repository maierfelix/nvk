Parameters object When instantiated with an object, members can be filled directly. Properties VkVertexInputAttributeDescription.prototype.location Number is the shader binding location number for this attribute. VkVertexInputAttributeDescription.prototype.binding Number is the binding number which this attribute takes its data from. VkVertexInputAttributeDescription.prototype.format VkFormat is the size and type of the vertex attribute data. VkVertexInputAttributeDescription.prototype.offset Number is a byte offset of this attribute relative to the start of an element in the vertex input binding. Default Properties VkVertexInputAttributeDescription.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkVertexInputAttributeDescription.memoryLayout Object Object describing the structure's memory layout. VkVertexInputAttributeDescription.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkVertexInputAttributeDescription.reset Resets the native memory of the structure. VkVertexInputAttributeDescription.flush Writes the structure's data into the native memory. VkVertexInputAttributeDescription.reflect Synchronizes the structure's data with the native memory.
