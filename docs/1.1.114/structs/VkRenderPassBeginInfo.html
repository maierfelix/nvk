Parameters object When instantiated with an object, members can be filled directly. Properties VkRenderPassBeginInfo.prototype.sType VkStructureType is the type of this structure. VkRenderPassBeginInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkRenderPassBeginInfo.prototype.renderPass VkRenderPass is the render pass to begin an instance of. VkRenderPassBeginInfo.prototype.framebuffer VkFramebuffer is the framebuffer containing the attachments that are used with the render pass. VkRenderPassBeginInfo.prototype.renderArea VkRect2D is the render area that is affected by the render pass instance, and is described in more detail below. VkRenderPassBeginInfo.prototype.clearValueCount Number VkRenderPassBeginInfo.prototype.pClearValues Array[VkClearValue] is an array of VkClearValue structures that contains clear values for each attachment, if the attachment uses a loadOp value of VK_ATTACHMENT_LOAD_OP_CLEAR or if the attachment has a depth/stencil format and uses a stencilLoadOp value of VK_ATTACHMENT_LOAD_OP_CLEAR. The array is indexed by attachment number. Only elements corresponding to cleared attachments are used. Other elements of pClearValues are ignored. Default Properties VkRenderPassBeginInfo.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkRenderPassBeginInfo.memoryLayout Object Object describing the structure's memory layout. VkRenderPassBeginInfo.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkRenderPassBeginInfo.reset Resets the native memory of the structure. VkRenderPassBeginInfo.flush Writes the structure's data into the native memory. VkRenderPassBeginInfo.reflect Synchronizes the structure's data with the native memory.
