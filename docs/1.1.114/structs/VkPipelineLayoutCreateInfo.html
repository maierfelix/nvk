VkPipelineLayoutCreateInfo.prototype.setLayoutCount Number is the number of descriptor sets included in the pipeline layout. VkPipelineLayoutCreateInfo.prototype.pSetLayouts Array[VkDescriptorSetLayout] is an array of VkDescriptorSetLayout objects. VkPipelineLayoutCreateInfo.prototype.pushConstantRangeCount Number is the number of push constant ranges included in the pipeline layout. VkPipelineLayoutCreateInfo.prototype.pPushConstantRanges Array[VkPushConstantRange] is an array of VkPushConstantRange structures defining a set of push constant ranges for use in a single pipeline layout.
