Parameters object When instantiated with an object, members can be filled directly. Properties VkPushConstantRange.prototype.stageFlags VkShaderStageFlagBits is a set of stage flags describing the shader stages that will access a range of push constants. If a particular stage is not included in the range, then accessing members of that range of push constants from the corresponding shader stage will return undefined values. VkPushConstantRange.prototype.offset Number VkPushConstantRange.prototype.size Number are the start offset and size, respectively, consumed by the range. Both offset and size are in units of bytes and must be a multiple of 4. The layout of the push constant variables is specified in the shader. Default Properties VkPushConstantRange.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkPushConstantRange.memoryLayout Object Object describing the structure's memory layout. VkPushConstantRange.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkPushConstantRange.reset Resets the native memory of the structure. VkPushConstantRange.flush Writes the structure's data into the native memory. VkPushConstantRange.reflect Synchronizes the structure's data with the native memory.
