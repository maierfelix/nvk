Parameters object When instantiated with an object, members can be filled directly. Properties VkBufferMemoryBarrier.prototype.sType VkStructureType is the type of this structure. VkBufferMemoryBarrier.prototype.pNext null is null or a reference to an extension-specific structure. VkBufferMemoryBarrier.prototype.srcAccessMask VkAccessFlagBits is a bitmask of VkAccessFlagBits specifying a <>. VkBufferMemoryBarrier.prototype.dstAccessMask VkAccessFlagBits is a bitmask of VkAccessFlagBits specifying a <>. VkBufferMemoryBarrier.prototype.srcQueueFamilyIndex Number is the source queue family for a <>. VkBufferMemoryBarrier.prototype.dstQueueFamilyIndex Number is the destination queue family for a <>. VkBufferMemoryBarrier.prototype.buffer VkBuffer is a handle to the buffer whose backing memory is affected by the barrier. VkBufferMemoryBarrier.prototype.offset BigInt Number is an offset in bytes into the backing memory for buffer; this is relative to the base offset as bound to the buffer (see vkBindBufferMemory). VkBufferMemoryBarrier.prototype.size BigInt Number is a size in bytes of the affected area of backing memory for buffer, or VK_WHOLE_SIZE to use the range from offset to the end of the buffer. Default Properties VkBufferMemoryBarrier.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkBufferMemoryBarrier.memoryLayout Object Object describing the structure's memory layout. VkBufferMemoryBarrier.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkBufferMemoryBarrier.reset Resets the native memory of the structure. VkBufferMemoryBarrier.flush Writes the structure's data into the native memory. VkBufferMemoryBarrier.reflect Synchronizes the structure's data with the native memory.
