Parameters object When instantiated with an object, members can be filled directly. Properties VkCopyDescriptorSet.prototype.sType VkStructureType is the type of this structure. VkCopyDescriptorSet.prototype.pNext null is null or a reference to an extension-specific structure. VkCopyDescriptorSet.prototype.srcSet VkDescriptorSet VkCopyDescriptorSet.prototype.srcBinding Number VkCopyDescriptorSet.prototype.srcArrayElement Number are the source set, binding, and array element, respectively. If the descriptor binding identified by srcSet and srcBinding has a descriptor type of VK_DESCRIPTOR_TYPE_INLINE_UNIFORM_BLOCK_EXT then srcArrayElement specifies the starting byte offset within the binding to copy from.
