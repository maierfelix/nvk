Parameters object When instantiated with an object, members can be filled directly. Properties VkDrawIndexedIndirectCommand.prototype.indexCount Number is the number of vertices to draw. VkDrawIndexedIndirectCommand.prototype.instanceCount Number is the number of instances to draw. VkDrawIndexedIndirectCommand.prototype.firstIndex Number is the base index within the index buffer. VkDrawIndexedIndirectCommand.prototype.vertexOffset Number is the value added to the vertex index before indexing into the vertex buffer. VkDrawIndexedIndirectCommand.prototype.firstInstance Number is the instance ID of the first instance to draw.
