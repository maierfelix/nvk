Properties VkVertexInputBindingDescription.prototype.binding Number is the binding number that this structure describes. VkVertexInputBindingDescription.prototype.stride Number is the distance in bytes between two consecutive elements within the buffer. VkVertexInputBindingDescription.prototype.inputRate VkVertexInputRate is a VkVertexInputRate value specifying whether vertex attribute addressing is a function of the vertex index or of the instance index. VkVertexInputBindingDescription.memoryLayout Object Object describing the structure's memory layout. VkVertexInputBindingDescription.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkVertexInputBindingDescription.reset Resets the native memory of the structure.
