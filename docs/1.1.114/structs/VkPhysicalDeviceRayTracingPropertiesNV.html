Properties VkPhysicalDeviceRayTracingPropertiesNV.prototype.sType VkStructureType is the type of this structure. VkPhysicalDeviceRayTracingPropertiesNV.prototype.pNext null is null or a reference to an extension-specific structure. VkPhysicalDeviceRayTracingPropertiesNV.prototype.shaderGroupHandleSize Number size in bytes of the shader header. VkPhysicalDeviceRayTracingPropertiesNV.prototype.maxRecursionDepth Number is the maximum number of levels of recursion allowed in a trace command. VkPhysicalDeviceRayTracingPropertiesNV.prototype.maxShaderGroupStride Number is the maximum stride in bytes allowed between shader groups in the SBT. VkPhysicalDeviceRayTracingPropertiesNV.prototype.shaderGroupBaseAlignment Number is the required alignment in bytes for the base of the SBTs. VkPhysicalDeviceRayTracingPropertiesNV.prototype.maxGeometryCount BigInt Number is the maximum number of geometries in the bottom level acceleration structure. VkPhysicalDeviceRayTracingPropertiesNV.prototype.maxInstanceCount BigInt Number is the maximum number of instances in the top level acceleration structure. VkPhysicalDeviceRayTracingPropertiesNV.prototype.maxTriangleCount BigInt Number is the maximum number of triangles in all geometries in the bottom level acceleration structure. VkPhysicalDeviceRayTracingPropertiesNV.prototype.maxDescriptorSetAccelerationStructures Number is the maximum number of acceleration structure descriptors that are allowed in a descriptor set.
