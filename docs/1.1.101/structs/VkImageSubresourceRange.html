Parameters object When instantiated with an object, members can be filled directly. Properties VkImageSubresourceRange.prototype.aspectMask Number is a bitmask of VkImageAspectFlagBits specifying which aspect(s) of the image are included in the view. VkImageSubresourceRange.prototype.baseMipLevel Number is the first mipmap level accessible to the view. VkImageSubresourceRange.prototype.levelCount Number is the number of mipmap levels (starting from baseMipLevel) accessible to the view. VkImageSubresourceRange.prototype.baseArrayLayer Number is the first array layer accessible to the view. VkImageSubresourceRange.prototype.layerCount Number is the number of array layers (starting from baseArrayLayer) accessible to the view. Default Properties VkImageSubresourceRange.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkImageSubresourceRange.memoryLayout Object Object describing the memory layout for this structure.
