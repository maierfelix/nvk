Parameters object When instantiated with an object, members can be filled directly. Properties VkPresentInfoKHR.prototype.sType VkStructureType is the type of this structure. VkPresentInfoKHR.prototype.pNext null is null or a reference to an extension-specific structure. VkPresentInfoKHR.prototype.waitSemaphoreCount Number is the number of semaphores to wait for before issuing the present request. The number may be zero. VkPresentInfoKHR.prototype.pWaitSemaphores Array[VkSemaphore] VkPresentInfoKHR.prototype.swapchainCount Number is the number of swapchains being presented to by this command. VkPresentInfoKHR.prototype.pSwapchains Array[VkSwapchainKHR] is an array of VkSwapchainKHR objects with swapchainCount entries. A given swapchain must not appear in this list more than once. VkPresentInfoKHR.prototype.pImageIndices Uint32Array is an array of indices into the array of each swapchain's presentable images, with swapchainCount entries. Each entry in this array identifies the image to present on the corresponding entry in the pSwapchains array. VkPresentInfoKHR.prototype.pResults Int32Array is an array of VkResult typed elements with swapchainCount entries. Applications that do not need per-swapchain results can use null for pResults. If non-null, each entry in pResults will be set to the VkResult for presenting the swapchain corresponding to the same index in pSwapchains. Default Properties VkPresentInfoKHR.prototype.memoryBuffer ArrayBuffer Memory representation of the original vulkan structure instance. VkPresentInfoKHR.memoryLayout Object Object describing the memory layout for this structure.
