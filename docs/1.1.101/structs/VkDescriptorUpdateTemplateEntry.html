Properties VkDescriptorUpdateTemplateEntry.prototype.dstBinding Number is the descriptor binding to update when using this descriptor update template. VkDescriptorUpdateTemplateEntry.prototype.dstArrayElement Number is the starting element in the array belonging to dstBinding. If the descriptor binding identified by srcBinding has a descriptor type of VK_DESCRIPTOR_TYPE_INLINE_UNIFORM_BLOCK_EXT then dstArrayElement specifies the starting byte offset to update.
