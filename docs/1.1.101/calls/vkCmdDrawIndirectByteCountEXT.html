 Categories vkCmdDrawIndirectByteCountEXT Draw primitives where the vertex count is derived from the counter byte value in the counter buffer Syntax 
void vkCmdDrawIndirectByteCountEXT();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command is recorded. instanceCount Number is the number of instances to draw. firstInstance Number is the instance ID of the first instance to draw. counterBuffer VkBuffer is the buffer handle from where the byte count is read. counterBufferOffset BigInt is the offset into the buffer used to read the byte count, which is used to calculate the vertex count for this draw call. counterOffset Number is subtracted from the byte count read from the counterBuffer at the counterBufferOffset vertexStride Number is the stride in bytes between each element of the vertex data that is used to calculate the vertex count from the counter value. This value is typically the same value that was used in the graphics pipeline state when the transform feedback was captured as the XfbStride.
