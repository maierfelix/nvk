 Categories vkCmdExecuteCommands Execute a secondary command buffer from a primary command buffer Syntax 
void vkCmdExecuteCommands();
Parameters commandBuffer VkCommandBuffer is a handle to a primary command buffer that the secondary command buffers are executed in. commandBufferCount Number pCommandBuffers Array[VkCommandBuffer] is an array of secondary command buffer handles, which are recorded to execute in the primary command buffer in the order they are listed in the array.
