 Categories vkGetRayTracingShaderGroupHandlesNV Query ray tracing pipeline shader group handles Syntax 
VkResult vkGetRayTracingShaderGroupHandlesNV();
Parameters device VkDevice is the logical device that contains the ray tracing pipeline. pipeline VkPipeline is the ray tracing pipeline object that contains the shaders. firstGroup Number is the index of the first group to retrieve a handle for from the VkPipelineShaderStageCreateInfo::pGroups array. groupCount Number is the number of shader handles to retrieve. dataSize Number is the size in bytes of the buffer pointed to by pData. pData ArrayBuffer is a reference to a user-allocated buffer where the results will be written.
